Crear y enviar una tarea

Una vez creada la tarea, podemos enviarla a trav√©s de una solicitud POST() al endpoint task. Esa solicitud debe ser autorizada con el token que obtuvimos en el paso anterior, y debe ser codificada como JSON (utilizamos toJSON() para transformar la tarea).
